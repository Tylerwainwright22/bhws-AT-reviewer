# bhws-AT-reviewer

A command-line tool that generates structured evaluations of assistive 
technology products for Black Hills Works clinical and vocational staff.

## What it does

Input an AT product name, get back a structured report covering:
- Product overview
- Population fit for adults with IDD
- Vocational relevance across employment phases
- South Dakota CHOICES waiver compatibility
- Data privacy considerations
- Staff recommendation (Explore / Investigate Further / Pass)
